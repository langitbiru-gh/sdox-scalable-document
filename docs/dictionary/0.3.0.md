# SDOX to HTML Dictionary

**Version:** 0.3.0  
**Purpose:** Standardized reference for translating SDOX v0.3.0 interactive syntax into semantic, rich, and stateful HTML equivalents.

---

## 🕹️ Interactive Elements Category

### `#question`
The container for a multiple-choice, single-choice, or true-false quiz item.
**HTML Equivalent:**
```html
<div class="sdox-question" id="{id}" data-type="{type}" data-state="unanswered">
  <!-- Inside contains question content (paragraphs, text) -->
  <div class="sdox-options-container">
    <!-- Option tags go here -->
  </div>
  <div class="sdox-explanation-wrapper hidden">
    <!-- Explanation tag goes here -->
  </div>
  <div class="sdox-question-actions">
    <button class="sdox-btn sdox-btn-reset" onclick="resetQuestion('{id}')">Reset Answer</button>
  </div>
</div>
```

### `#option`
A selectable choice within a question.
**HTML Equivalent:**
```html
<button class="sdox-option" 
        data-key="{key}" 
        data-correct="{correct}" 
        data-explanation="{explanation}"
        onclick="selectOption('{question_id}', '{key}')">
  <span class="sdox-option-marker">{key}</span>
  <span class="sdox-option-label"><!-- content --></span>
</button>
```

### `#explanation`
A rich feedback block revealed once a question is answered.
**HTML Equivalent:**
```html
<div class="sdox-explanation">
  <div class="sdox-explanation-header">
    <span class="icon">💡</span> Explanation
  </div>
  <div class="sdox-explanation-body">
    <!-- content -->
  </div>
</div>
```

---

### `#flashcard`
A two-sided study card that flips to show the answer when clicked.
**HTML Equivalent:**
```html
<div class="sdox-flashcard" id="{id}" onclick="this.classList.toggle('flipped')">
  <div class="sdox-flashcard-inner">
    <div class="sdox-flashcard-front">
      <!-- front content goes here -->
    </div>
    <div class="sdox-flashcard-back">
      <!-- back content goes here -->
    </div>
  </div>
</div>
```

### `#front` / `#back`
The front and back panels of a flashcard.
**HTML Equivalent:**
These are directly parsed into their respective card side slots as shown in the `#flashcard` structure.

---

### `#stepper`
A multi-step container for guides, tutorials, or step-by-step wizard flows.
**HTML Equivalent:**
```html
<div class="sdox-stepper" id="{id}" data-current-step="0" data-linear="{linear}">
  <div class="sdox-stepper-header">
    <!-- Generated step markers based on nested #step elements -->
    <div class="sdox-step-indicator active" data-step="0">1. Step Name</div>
  </div>
  <div class="sdox-stepper-body">
    <!-- Nested step blocks, showing only the active index -->
  </div>
  <div class="sdox-stepper-actions">
    <button class="sdox-btn sdox-btn-prev disabled" onclick="stepperPrev('{id}')">Back</button>
    <button class="sdox-btn sdox-btn-next" onclick="stepperNext('{id}')">Next</button>
  </div>
</div>
```

### `#step`
An individual step panel in a stepper guide.
**HTML Equivalent:**
```html
<div class="sdox-step-content" data-step-title="{title}" data-optional="{optional}">
  <!-- content -->
</div>
```

---

### `#poll`
An interactive simulator counter for live, multi-vote classroom voting sessions (e.g. class president elections).
**HTML Equivalent:**
```html
<div class="sdox-poll" id="{id}">
  <h3 class="sdox-poll-title">{question}</h3>
  <div class="sdox-choices-container">
    <!-- Choice elements go here -->
  </div>
  <div class="sdox-poll-actions">
    <button class="sdox-btn sdox-btn-reset-poll" onclick="resetPoll('{id}')">Reset Poll</button>
  </div>
</div>
```

### `#choice`
An individual selectable voting option inside a poll. Clicks increment its tally instantly.
**HTML Equivalent:**
```html
<button class="sdox-choice" data-key="{key}" data-votes="{votes}" onclick="voteChoice('{poll_id}', '{key}')">
  <div class="sdox-choice-bar" style="width: {percentage}%;"></div>
  <div class="sdox-choice-content">
    <span class="sdox-choice-label">{label}</span>
    <span class="sdox-choice-stats"><strong class="votes-count">{votes}</strong> votes ({percentage}%)</span>
  </div>
</button>
```

---

### `#random-picker`
An interactive spin-wheel, card shuffler, or lucky draw raffle widget.
**HTML Equivalent:**
```html
<div class="sdox-random-picker picker-{type}" id="{id}">
  <div class="sdox-picker-visualizer">
    <!-- Under spin-wheel type, renders a Canvas / SVG spinning wheel of options -->
    <div class="wheel-canvas-container">
      <canvas class="sdox-wheel-canvas"></canvas>
      <div class="wheel-pointer">▼</div>
    </div>
  </div>
  <div class="sdox-picker-winner hidden">
    Selected: <span class="winner-name">-</span>
  </div>
  <div class="sdox-picker-actions">
    <button class="sdox-btn sdox-btn-trigger-picker" onclick="spinWheel('{id}')">SPIN!</button>
  </div>
</div>
```

### `#picker-option`
A raffle/spin-wheel entry in a random picker.
**HTML Equivalent:**
Renders as a slice in the visual spin-wheel canvas or a card, weighted according to `{weight}`.
