# SDOX to HTML Dictionary

**Version:** 0.2.0  
**Purpose:** Standardized reference for translating SDOX v0.2.0 syntax into Semantic HTML.

---

## 📊 Visualization Category

### `#chart`
A container for data visualization. Recommendation: Use libraries like Chart.js or D3.js.
**HTML Equivalent:**
```html
<div class="sdox-chart-container">
  <canvas class="sdox-chart" data-type="{type}" data-title="{title}"></canvas>
</div>
```

### `#data`
Data points for a chart. Usually parsed and passed to a JS charting library.
**Standalone Fallback:** Render as a progress bar or metric badge.
**HTML Equivalent:**
```html
<div class="sdox-data-fallback">
  <span class="label">{label}</span>
  <progress value="{value}" max="100"></progress>
</div>
```

## ∑ Mathematics Category

### `#math`
Mathematical formulas. Recommendation: Use KaTeX or MathJax for rendering.
**HTML Equivalent:**
```html
<div class="sdox-math {display}">
  <!-- Raw LaTeX content -->
</div>
```

## 🎨 Diagrams Category

### `#diagram`
Structural diagrams. Recommendation: Use Mermaid.js.
**HTML Equivalent:**
```html
<div class="mermaid sdox-diagram">
  <!-- Raw diagram notation -->
</div>
```

### `#timeline`, `#event`
Chronological event display.
**Standalone Fallback (`#event`):** Render as a standalone milestone card.
**HTML Equivalent:**
```html
<div class="sdox-timeline timeline-{orientation}">
  <div class="event-item">
    <div class="event-date">{date}</div>
    <div class="event-content">
      <h4 class="event-title">{title}</h4>
      <div class="event-body"><!-- content --></div>
    </div>
  </div>
</div>
```

## 🧭 Navigation & Referencing Category

### `#toc`
Table of Contents. Usually generated dynamically from document headings.
**HTML Equivalent:**
```html
<nav class="sdox-toc">
  <ul><!-- generated links --></ul>
</nav>
```

### `#ref`
Internal cross-reference.
**HTML Equivalent:**
```html
<a href="#{to}" class="sdox-ref">[{to}]</a>
```

### `#footnote`
A footnote reference and content.
**HTML Equivalent:**
```html
<sup class="sdox-footnote-ref" id="fn-ref-{id}"><a href="#fn-{id}">{id}</a></sup>
...
<div class="sdox-footnote-content" id="fn-{id}">
  <a href="#fn-ref-{id}">↩</a> <!-- content -->
</div>
```

---

## 📐 Layout Category

### `#grid`, `#column`
Multi-column layout.
**Standalone Fallback (`#column`):** Render as a bordered content block.
**HTML Equivalent:**
```html
<div class="sdox-grid" style="grid-template-columns: repeat({columns}, 1fr);">
  <div class="sdox-column" style="grid-column: span {span};">
    <!-- content -->
  </div>
</div>
```

### `#video`
Embedded video player.
**HTML Equivalent:**
```html
<div class="sdox-video-wrap">
  <video src="{src}" controls="{controls}" class="sdox-video"></video>
</div>
```

### `#audio`
Embedded audio player.
**HTML Equivalent:**
```html
<div class="sdox-audio-wrap">
  <audio src="{src}" controls="{controls}" class="sdox-audio"></audio>
</div>
```

### `#badge`
Small status indicator.
**HTML Equivalent:**
```html
<span class="sdox-badge badge-{type}">{content}</span>
```

### `#html`
Render raw HTML content.
**HTML Equivalent:**
```html
<div class="sdox-html-render">
  <!-- Raw HTML content -->
</div>
```

### `#markdown`
Render parsed Markdown content.
**HTML Equivalent:**
```html
<div class="sdox-markdown-render">
  <!-- Parsed Markdown HTML -->
</div>
```

## 📚 Documentation Category

### `#term`, `#definition`
Glossary entries.
**HTML Equivalent:**
```html
<dl class="sdox-glossary">
  <dt id="{id}" class="sdox-term">{content}</dt>
  <dd class="sdox-definition" for="{for}"><!-- content --></dd>
</dl>
```

### `#changelog`
Version history section.
**HTML Equivalent:**
```html
<div class="sdox-changelog">
  <div class="changelog-header">Change Log</div>
  <div class="changelog-body">
    <!-- content -->
  </div>
</div>
```
